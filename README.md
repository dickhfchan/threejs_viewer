
## Run mode

- (Pack all) Development 
- (Pack all) Build 
- (Library) Development
- (Library) Build


### (Pack all) Development

```
npm i
npm run serve
```

### (Pack all) Build

```
npm i
npm run build
```

### (Library) Development

```
npm i
npm run serve-lib
```


### (Library) Build

```
npm i
npm run build-lib
```
